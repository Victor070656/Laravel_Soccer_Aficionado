# Mobile App Screen Ideas - Soccer Aficionado

## Bottom Navigation Tabs (5-Tab Structure)

### 1. Home (Football Feed)
- **Header**: Glassmorphism card with user points/badges
- **Post Type Selector**: 8 content types with icons (banter 💬, match_reaction ⚽, etc.)
- **Feed Cards**: Glassmorphism with content type badges, quick reactions on hover
- **Sidebar**: Live matches widget, active polls, trending topics

### 2. Trending (Twitter/X-like)
- **Breaking News Ticker**: Red accent bar, hot badge, auto-scrolling
- **Trending Hashtags**: #ArtetaOut, #HalaMadrid, etc. with post counts
- **Viral Debates**: High-engagement posts with type badges
- **Sidebar**: Most discussed clubs (top 5), trending players, active match rooms

### 3. Communities (Club Ecosystem)
- **Location Filter Tabs**: All | Global | By Country | By State/Region
- **Community Cards**: Club logo, member count, post count, location badge
- **Search Bar**: Instant filtering of communities
- **Sidebar**: Popular clubs list for quick community creation

### 4. Match Rooms (Live Banter)
- **Match Header**: Teams + score, live pulse, heat meter bar
- **Comment Input**: Quick post with emoji suggestions
- **Emoji Buttons**: 7 reactions (⚽, 🔥, 💚, 😂, 😡, 🎉, 👏)
- **Heat Meter**: Color-coded (CALM=outline → LOW=secondary → MEDIUM=tertiary-fixed-dim → HIGH=primary-container → EXTREME=error)
- **Fan Momentum Bar**: Gradient showing home vs away support
- **Live Comments**: User avatars, club badges, scrollable feed
- **Match Events Ticker**: Goals, cards, subs with icons
- **Emoji Storm Overlay**: Floating emojis when storm detected

### 5. Profile (Football Identity Card)
- **Profile Header**: Avatar with club badge overlay, stadium glow effects
- **Stats Row**: Points, Followers, Following, Posts
- **Football Identity**: Favorite player/coach sections with photos
- **Fan Ranking**: Legendary Fan, Elite Fan, Super Fan, etc. with progress bar
- **Achievements**: Badge grid with icons
- **Recent Activity**: Posts and comments sections
- **Top Communities**: Club communities with role badges

## Screen Flow Examples

### Matchday Flow
1. Open app → Bottom nav shows "3 Live" badge on Match Rooms
2. Tap Match Rooms → See live matches with heat indicators
3. Tap a match → Enter room with live comments streaming
4. Post reaction → "⚽ Goal!" → See emoji storm trigger
5. Heat meter rises to "EXTREME" → Feel the stadium energy

### Daily Engagement Flow
1. Open app → Home feed with latest banter
2. Scroll → See match_reaction post → Quick react with "🔥"
3. Tap trending → See #ArtetaOut debate → Join conversation
4. Check profile → See fan ranking improved → Feel recognized
5. Visit communities → Join Arsenal community → Gain club loyalty badge
