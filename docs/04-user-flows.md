# User Flow Concepts - Soccer Aficionado

## Core User Journeys

### 1. New Fan Onboarding
```
Landing Page → Register → Select Favorite Clubs (max 3) → Choose Football Personality
(Flame, Analyst, Banter King, etc.) → Get "New Fan" Ranking → Redirected to Home Feed
```

### 2. Daily Engagement Loop
```
Open App → Check Live Matches (bottom nav badge) → Scroll Home Feed → React to Post
(quick emoji) → Comment → Check Trending → Join Debate → Visit Profile
→ See Points Increased → Feel Rewarded
```

### 3. Matchday Experience
```
Notification: "Match Starting" → Open Match Room → See Live Score + Heat Meter
→ Post "⚽ Come on!" → See Others Reacting → Emoji Storm Triggers
→ Goal Scores → Comments Explode → Heat Meter → "EXTREME" → Celebrate
→ Check Fan Momentum → See Home Support Rising → Feel Tribal Loyalty
```

### 4. Community Discovery
```
Bottom Nav → Communities Tab → See Location Filters (All/Global/Country/State)
→ Search "Arsenal" → Find "Arsenal Community" → See 5,000 Members
→ Read Discussions → Join Community → Get "Club Loyalty" Badge → Post Introduction
→ Get Welcome Comments → Feel Belonging
```

### 5. Profile Building
```
Visit Own Profile → See Football Identity Card → Add Favorite Player/Coach
→ Upload Avatar → Post Banter → Get Likes → Points Increase
→ Unlock "Rising Fan" Ranking → See Progress Bar → Post More → Unlock Badges
→ Check Fan Ranking → "Active Fan" → Feel Progress → Share Profile → Gain Followers
```

### 6. Viral Debate Participation
```
Home Feed → See "Viral Debate" Badge → Tap → Read Tactical Opinion
→ Comment with Counter-Argument → Get 10 Likes → Featured in Trending
→ Check Profile → "Debate Won" Badge Earned → Feel Recognition → Notification:
"Your debate is trending!" → More Engagement
```

## Retention Loops

### The Points Loop
1. **Action**: Post comment → +5 points
2. **Feedback**: Points counter animates up
3. **Reward**: Reach 100 points → "Rising Fan" ranking
4. **Recognition**: Profile shows new ranking
5. **Motivation**: Want "Active Fan" (500 points) → Post more

### The Club Loyalty Loop
1. **Join**: Join club community
2. **Participate**: Post in community → Get club-specific reactions
3. **Milestone**: 10 posts for club → "Club Loyalty" badge
4. **Status**: Profile shows club badge overlay
5. **Pride**: Display loyalty → Attract followers → Grow reputation

### The Matchday Streak Loop
1. **Check-in**: Post during match → +5 points
2. **Streak**: 3 matches in a row → "Matchday Streak" badge
3. **Community**: Others see your streak → Respect from fans
4. **Competition**: Try to maintain streak → Daily app opens
5. **Habit**: Open app every matchday → Core user behavior
